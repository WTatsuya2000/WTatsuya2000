    液体シンチレータのn/gamma PSDの手法準備
    rootの使い方を思い出す必要がある。

    mkdocs

    これは完全に忘れていた。GitHubの使い方もなんとなくわかってきたが、アカウントの使い分けには注意が必要そうである。

    mkdocs gh-deploy

    基本はこれ。これでエラーが出る場合は2つくらい可能性あり。

    • トークンの期限切れ githubの行って更新する。developerのとこから入る。最近新しい方法がさらに出てるらしい。

    • アカウントとpush先の入れ違い
      これがめんどかった。

    ~/.gitconfig

    の編集でいける。(mac)特定のとこだけ別垢を使うなら

    ~/.gitconfig_@@@@

    に書き込めばいい。 ついでに

    git config --list
git config --local -e

    も確認すると大体解決するはず
